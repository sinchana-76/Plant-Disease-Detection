🌿 Plant Disease Detection Using CNN
📌 Project Overview

This project is a Plant Disease Detection Web Application built using Deep Learning and Flask.
The system uses a Convolutional Neural Network (CNN) to analyze plant leaf images and detect diseases.

Users can upload a plant leaf image, and the model predicts whether the plant is healthy or affected by a disease.

📥Download Trained Model:

The trained CNN model is hosted on Kaggle.

[Download Trained CNN Model](https://www.kaggle.com/datasets/sinchanagsingh/plant-disease-detection-cnn-model)

After downloading, keep the model file in the same folder as app.py.

Example:
project_folder
│
├── app.py
├── plant_disease_prediction_model.h5

📊 Dataset:
This project uses the PlantVillage dataset, which contains thousands of labeled images of healthy and diseased plant leaves.
